¿Qué es una Red?

Una red informática es un conjunto de dispositivos interconectados entre sí para compartir recursos y datos. Se utilizan diferentes tipos de redes, entre las cuales están:

  • LAN (Red de Área Local): Conecta dispositivos en una ubicación física limitada, como una casa o una oficina.
  • WAN (Red de Área Amplia): Conecta redes más amplias, como la conexión entre ciudades o incluso países.
  • Wi-Fi: Red inalámbrica que utiliza ondas de radio para conectar dispositivos sin necesidad de cables.

Las redes permiten el intercambio eficiente de información y el acceso compartido a recursos como archivos e impresoras.
